# 🐍 Snake Game in Python

A classic Snake Game developed in Python using the Turtle Graphics library.

## Features

- Smooth snake movement
- Food spawning
- Score tracking
- Collision detection
- Game Over screen
- Keyboard controls (W, A, S, D)

## Tech Stack

- Python
- Turtle Graphics

## Controls

| Key | Action |
|-----|--------|
| W | Move Up |
| A | Move Left |
| S | Move Down |
| D | Move Right |

## Installation

Clone the repository:

```bash
git clone https://github.com/Devang3105/Snake_game.git
```

Run the game:

```bash
python snake.py
```
